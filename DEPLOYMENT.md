# Guide de Déploiement - Portfolio Henrik Velnom

## Options d'Hébergement et URLs

### 1. **Netlify** (Recommandé - Le plus simple) ⭐
- **URL par défaut**: `https://henrik-velnom-portfolio.netlify.app`
- **URL personnalisée possible**: `https://henrikvelnom.com` (si vous avez un domaine)
- **Avantages**: 
  - Gratuit
  - Déploiement en drag & drop
  - HTTPS automatique
  - Domaine personnalisé facile

**Comment déployer:**
1. Aller sur https://www.netlify.com
2. Créer un compte (gratuit)
3. Glisser-déposer le dossier du projet dans Netlify
4. Le site sera en ligne en quelques secondes
5. Vous pouvez personnaliser l'URL dans les paramètres

---

### 2. **Vercel** (Alternative excellente)
- **URL par défaut**: `https://henrik-velnom-portfolio.vercel.app`
- **Avantages**:
  - Gratuit
  - Très rapide
  - Déploiement via GitHub ou drag & drop

**Comment déployer:**
1. Aller sur https://vercel.com
2. Créer un compte
3. Glisser-déposer le dossier du projet
4. Site en ligne immédiatement

---

### 3. **GitHub Pages** (Si vous avez GitHub)
- **URL**: `https://[votre-username].github.io/henrik-portfolio/`
- **Exemple**: `https://henrikvelnom.github.io/henrik-portfolio/`
- **Avantages**:
  - Gratuit
  - Intégration avec GitHub
  - HTTPS automatique

**Comment déployer:**
1. Créer un repository sur GitHub
2. Uploader tous les fichiers
3. Aller dans Settings > Pages
4. Sélectionner la branche main
5. Le site sera disponible à l'URL GitHub

---

### 4. **Domaine Personnalisé** (Optionnel)
Si vous avez un domaine (ex: `henrikvelnom.com`):
- Vous pouvez l'ajouter à Netlify, Vercel, ou GitHub Pages
- Configuration DNS nécessaire
- Coût du domaine: ~10-15€/an

---

## Fichiers Prêts pour le Déploiement

✅ Tous les fichiers sont prêts:
- `index.html` - Page principale
- `styles.css` - Styles
- `script.js` - Interactions
- `translations.js` - Système multilingue
- Toutes les images
- Favicons (HV)

## Recommandation

**Netlify** est la meilleure option car:
1. Déploiement le plus simple (drag & drop)
2. URL personnalisable facilement
3. HTTPS automatique
4. Performance excellente
5. Interface intuitive

**URL suggérée sur Netlify:**
- Par défaut: `henrik-velnom-portfolio.netlify.app`
- Vous pouvez la changer en: `henrik-velnom.netlify.app` ou autre

## Étapes Rapides pour Netlify

1. Aller sur https://app.netlify.com/drop
2. Glisser le dossier du projet
3. Copier l'URL générée
4. (Optionnel) Personnaliser l'URL dans Site settings > Change site name
